# 📷Image resize square

Make the image as ratio square.

どんな画像も正方形にします。

## Demo

![demo](.\demo.gif)

## Usage

Place this code in an environment where Python is running.
Just D&D the image you want to make a square.

このコードを Python が動く環境に配置して、正方形にしたい画像を D&D するだけです。
